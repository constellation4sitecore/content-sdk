# Constellation SXA NextJS for Content SDK

## Installation

```bash
npm install @constellation4sitecore-content-sdk/nextjs --save
```

## Setup Serialization

In your project add in `sitecore.json`

```json
  "modules": [
    ....
    "npm:@constellation4sitecore-content-sdk/nextjs"
  ],
```

Then you can push the items by

```bash
dotnet sitecore ser push
```
